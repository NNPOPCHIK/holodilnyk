# 🥶 Telegram-бот «Холодильник»

Простой Telegram-бот для учета продуктов и контроля сроков годности.  
Позволяет добавлять продукты, просматривать список, проверять просрочку и удалять записи с подтверждением.

---

## 🚀 Возможности

- ➕ Добавление продуктов (пошаговый ввод: название → день → месяц → год)
- 📋 Просмотр списка продуктов
- ⚠️ Проверка просроченных продуктов
- 🗑 Удаление продуктов с подтверждением
- 🌐 Поддержка SOCKS5 прокси
- 💾 Хранение данных в JSON-файле

---

## 📦 Установка и запуск

### 1. Скачивание проекта
Скачай проект как ZIP-архив с GitHub и распакуй его.

---

### 2. Переход в папку проекта через cmd
```bash
cd holod
```
и выполни:
```bash
pip install -r requirements.txt
```
### 3. После чего возьми API_TOKEN в боте BotFather
и вставь в строку bot.py
```bash
TOKEN = "your_token"
```
### 4. Теперь можешь запускать бота
```bash
python bot.py
```

Также из-за возможных ограничений с доступом к телеграму добавленна возможность добавить собственный прокси типа Socks5 в строке форматом
```bash
PROXY = "socks5://user:pass@ip:port"
```
или же без авторизации
```bash
PROXY = "socks5://ip:port"
```
